Exemplo de cores

Podemos representar as cores em CSS de qutro formas:

  • Por nome: Escrevendo o nome da cor desejada (em inglês). Ex. red, white, black etc.
  • Código exadecimal: Combinação de números e letras que variam de 0 a 10 e da letra "a" ate'a letra "f"

Criando Degradê

Colocamos o Degradê no site por meio do CSS, neste código o mesmo está sendo feito localmente, ouseja, configurado no cabeçalho do site (HEAD), Abrimos um STYLE e adicionamos,por exemplo BODY, para alterar a cor de fundo em Degradê e depois usamos a tag BACKGROUND-IMAGEM e depois selecionamos LINEAR-GRADIENTE, onde escolhemos o sentido do Degradê e depois da cor inicial a cor final. Segue exemplo do código: 

            
               
                    body {
            
                        background-image: linear-gradient(to bottom, white, blue);
            
            
                    }

Ainda é possível fazer o Degradê variar mais de uma vez, por exemplo: de azul, para branco e depois voltar novamente para o azul.

A mesma técnica pode ser feita com exadecimal: exemplo: 

            
                body {
            
                    background-image: linear-gradient(to bottom, #3C7A71, #26C7AF); 
                    
                    
                }

É possível usar o Adobe color ou outra ferramenta do gênenro para extrair o código da cores.

Assim como as cores nominais, podemos colocar quantas cores quisermos.

Podemos direcionar as cores por ângulos em vez de direções (to top, bottom, right, left), assim usamos o valor do ângulo e a trerminação DEG. Vejo a seguir: 

            
                body {
        
                    background-image: linear-gradient(45deg, #3C7A71, #26C7AF,#ffffff);
        
        
                }
As cores irão ficar em degradê num ângulo de 45 graus.

Dentro do STYLE, ainda podemos configurar o BODY com a BACKGROUND- ATTACHMENT e selecionarmos FIXED, isso fará que o degradêse fixe na página, não variando com a rolagem. vejo o código. 

            
                background-attachment: fixed;

Na função BACKGROUND-IMAGEM podemos selecionar o RADIAL-GRADIENT e vez de direcionamento ou ângulos e podemos escolher o formato do degradê, por exemplo CIRCLE. 
        
            background-image: radial-gradient(cirlce, white,blue);

Ainda podemos configurar a quantidade da cor que aparece no site através de porcentagem, colocamos a porcentagem imediatamente após a cor.

código: 

            
                body {
            
                    background-image: linear-gradient(45deg, #3C7A71 10%, #26C7AF 30%,#ffffff); 
                    background-attachment: fixed;
                    
                }

Importante

Importante lembrar que o ideal é a paleta conter entre 3 e 5 cores.
