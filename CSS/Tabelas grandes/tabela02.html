Ainda no código, dentro do TH, devemos adicionar o scope, o qual ajuda os mecanismos de buscas do Goolge a analisar sua tabela.

Quando falamos de configuração de linhas da tabela, para a melhor visualização podemos usar o efeito zebrado e pra isso estlizamos as linha dentro do TBODY, porém usando uma pseudoclassse NTH CHILD (). Dentro dos parênteses colocamos o intervalo do zebramento das linhas, no caso do código 2 em 2 para que a tabela fique zebrada. Sde por acaso, em vez de o núemro de colocarmos o número de linhas que queremos o intervalo colocarmos ODD (ímpar) serão configuradas todas as linhas com numeração ímpar. O mesmo serve se colocarmos EVEN (par).

Vamos colocar cabeçalho fixo na tabela: Primero vamos colocar a tabela em posição relativa. (position:relative), depois estilizamos o TH, dentro do TR dentro do TBODY coo posição STICKY, DEPOIS USAMOS TOP: 0 e configuramos o background novamente com a cor que o mesmo estava anteriormente.


        

        tbody > tr:nth-child(2n) {
