Teste de imagem de fundo

Neste teste, como novidade, temos o uso do BACKGROUND-SIZE: COVER. Esta função vai fazer com que a imagem fique cobrindo a tela 100%, mas ela fará uma adaptação para isso, podemos usar também, o CONTAIN para cobrir a tela, porém quando diminuida, a imagem ficará com barras laterais.

Comentários: pode também selecionar as linhas ou deixar o cursor em uma linha específica que será comentada e pressionar o Ctrl + K + C pra comentar e o Ctrl + K + U pra "descomentar"
