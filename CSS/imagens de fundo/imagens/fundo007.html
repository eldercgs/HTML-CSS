O conteúdo é relative e o container é absolute. Quando se configura dessa maneira é possível usar duas propriedades: LEFT e TOP.

TOP e LEFT podem ser prenchidos tanto em pixels quanto em porcentage.

Utilizamos a propriedade TRANSFORM: TRANSLATE () para movimentar o conteúdo, quando colocamos números positivos a caixa se movimenta para baixo e para esquerda e quando colocamos números negativos a caixa se posiciona para cima e para a direita. Assim, podemos movimentar a caixa para posicion-la onde for necessário, no caso, centralização.
