Tipos de alinhamento de texto

Existem 3 tipos de alinhamentos: à esquerda, À direita e centralizado.

As atribuição que usamos para alinhar os textos são as propriedades de CSS: 

        text-align: justify;
        text-align: center;
        text-align: righ;
        text-align: left;

a atribuição TEXT-INDENT permiti que deixemos um espaço inicial da linha para fazermos o parágrafo.
