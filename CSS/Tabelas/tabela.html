Tabela

Todas as tabelas simples em HTML tem a seguinte hierarquia: 

            TABLE = tabela
                ROW = linha de tabela
                    TABLE HEADER = cabeçalho de tabela
                    TABLE DATA = dado de tabela

A1 B1 C1
A2 B2 C2
A3 B3 C3
A4 B4 C4

O atributo [border-collapse: collapse] serve para unir as células das tabelas, ele vem padrão saparate.

Em HTML5 o fechamento ta tag TD é opcional.

O atributo[ vertical-align: top], serve para alinhar verticalmente a célula, o padrão é middle (meio), também podemos usar [bottom] (embaixo).
