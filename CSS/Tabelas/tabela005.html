Destacando colunas

A fim de destacar as colunas de uma tabela, dentro da tabela criamos a tabela COLGROUP e dentro desta, criamos a tag col, o número de COL depende do número de colunas. Depois só estilizar os COLs coo se fosse uma class.

        
            colgroup>
                col class="nome">
                col class="sexo">
                col class="idade">
                col class="prof">
            /colgroup>
        
    
        
            col.nome {
                background-color: aqua;
            }
            col.sexo {
                background-color: aqua;
            }
            col.idade {
                background-color: aqua;
            }
            col.prof {
                background-color: aqua;
            }

Podemos optar também, por colocar uma tag span "" dentro do COL, onde o número dentro das aspas representa o número de colunas que aquela configuração vai atingir.

        
            colgroup>
                col class="nome">
                col class="cgroup span "2"
                col class="prof">
            /colgroup>
        
    
        style {
            .colgroup {
                background-color: yellow;
            }
        }
