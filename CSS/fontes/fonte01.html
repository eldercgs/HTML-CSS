Trabalhando com fontes

Pode-se configurar a fonte do site por meio do CSS solicitando uma fonte de maneira genérica, por exemplo: Com serifa ou sem serifa ou monoespaçada simplismente. 

            
            style
                body{
                    font-family: sans-serif;
                }
            
            style

É possível, também usar a família das fonte: 

            
                body{
                    font-family: Arial, Helvetica, sans-serif
                }

Pesquisa de fontes

Podemos pesquisar fontes, suas famílias e combinações no site w3schools.com.

Tamanho das fontes

Temos as medidas absolutas e as medidas relativas

Medidas absolutas

São medidas absolutas:

  • cm, mm, in, px, pt (medida tipográfica, é utilizada no Word, por exemplo), pc

Medidas Relativas

  • em, fx, rem, vw, vh, %

Essas são algumas, mas existem outras.

O W3C recomenda, para o uso de fonte, utilizar as medidas "em" ou "px".

16px, gerelamente é o tamanho normal da fonte. 16px, também, geralmente, é equilavelnte a 1em.

Em caso de dúvidas, podemos acessar o site nekocalc.com para fazermos a conversão.

Peso das fontes

Para configurar o peso das fontes usamos a tag FONT-WEIGHT. Para esta ação, podemos configurar com algumas opções pré configuradas: lighter, normal, bold, e bolder, ou podemos usar uma medida numérica que varia de 100 a 900. É possível usar o atalho CONTROL + ESPAÇO que aparecerão as opções.

Estilo da fonte

Usamos a tag FRONT-STYLE, para configuraro estilo das fontes. É importante lembrar que nem todoas as fontes tem todas as opções de estilo e peso.

Shorthand

Épossível simplificar as configurações de fontes através de shorthands ("abreviações"), no caso das fontes, há uma ordem a ser seguida na configuração:
  1. font-style
  2. font-weight
  3. font-size
  4. font-family

A seguir exemplo:

        
            body{
                
                 font: italic bolder 1em arial;
                 
                }

Em vez de usar 4 linhas para codar, usa-se só uma linha. A seguir exemplo do código sem shorthand:

        
            font-style: italic;
            font weight;
            font-size: 16px ;
            font-family: 'work sans',Arial, Helvetica, sans-serif;

Não existe Shorthand para tudo, mas exisem muitos tipos de shorthands que podem ajudar no código. Pode-se usar o shorthand somente com as tags que são necessárias, porém as tags preisam estar na ordem pré estabelecida. Exemplo


        font: bolder 1em times New Roman;
