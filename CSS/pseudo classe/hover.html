Página 01

Podemos configurar elementos dentro das DIVs, quando escritos no CSS utilizamos o sinal ">" para indicar que pertence a DIV. Exemplo:

        div > p {
            display: none;
        }

Significa que o parágrafo dentro da DIV, quando passado o mouse em cima, terá a seguinte reação, no caso, mostrar o texto escondido pela primeira configuração da DIV.
