Exemplo de caixa box-level

Parágrafos também são exemplos de BOX-LEVEL, mas os links são exemplos de caixas inline-level. Vamos ver como tudo isso funciona.

Para melhor compreensaão dos exemplos, é necessário verificar o código do exercício. Trabalhamos com os conceitos e configurações de margem, outline e padding.

Usando shorthands

Podemos usar alguns shorthands para configurar as caixas (box-leve):

        border-width: 10px;
        border-style: solid;
        border-color: darkblue;

O código acima é o mesmo que:

        border: 10px, solid, darkblue;

Outros shorthands

O padding, tambémpode ser configurado em shorthand:

        padding-top: 10px;
        padding-right: 10px;
        padding-bottom: 10px;
        padding-left: 10px;
    
        padding: 10px (top), 10px (right), 10px (bottom), 10px (left);

Coloquei a ordem de colocação das medidas, mas não pode colocar os parênteses. Configura-se em sentido horário.

Ainda podemos simplificar ainda mais nosso código, pois se o valor de todas as margens ou paddings for o mesmo, por exemplo 10px, basta indicar o valor somente uma vez e o programa entenderá que as configurações de todos os lados é a mesma. Exemplo:

padding: 10px;
border:10px;

Outro maneira de se configurar elementos como margem, padding ou outline é indicando apenas dois valores, o programa entenderá que o primeiro valor é a configuração da parte de cima e de baixo (top, bottom) e o segundo número a configuração dos lados direito e esquerdo (right e left). Exemplo:

padding: 10px 20px;

No exemplo acima, sem shorthand ficaria assim:

        padding-top: 10px;
        padding-right: 20px;
        padding-bottom: 10px;
        padding-left: 10px;

Para centralizar a margem com o objetivo de centralizar a caixa no meio da página, usamos MARGIN: AUTO;

Podemos mesclar esse comando com o shorthand exemplificado acima:

        margin 10px auto 40px auto;

para OUTLINE usamos a seguinte configuração como shorthand: -width, -style, -color. Veja o exemplo:

        outline: 10px solid red;

Os shorthands diminuem consideravelmente seu código.

Configurando o display

a configuração padrão para a box-level é DISPLAY BLOCK e para o box-inline é DISPLAY INLINE, porém podemos mudar essa configuração padrão através do simples comando dentro do H1:

        display: inline;

Podemos ver que é bem diferente de como se apresenta a cima, no topo do site, onde a configuração permanece padrão.

Sombra na caixa

Podemos colocar sombra nas caixas usando o atributo BOX-SHADOW, configuramos ele declarando 4 atributos: espaçamento horizontl, espaçamento vertical,espalhamento e cor da sombra.

Podemos fazer uma pré visualização ou pré edição através da função inspecionar do site. Basta clicar com o botão direito e selecionar inspecionar.

O elemento INSET serve para fazer sombra dentro da caixa. Deixei uma configuração de maneira exagerada no NAV > A para destacar o efeito.
