Pseudo-classes definem estados de elementos em HTML.

Pseudo-elementos mexem diretamente no conteúdo periférico do elemento. Podemos ativa-los colocando duas vezes :: depois da tag desejada. Veja exemplo:

        a::after{
            content: ' [Link]'; /*Adiciona conteúdo após o link*/
            text-decoration: none;
            font-weight: normal;
            color: blue;}

É possível, também, usar uma combinação de pseudo-classe e pseudo-elemento:

        a:hover::after{
            font-weight: bolder;
            color: black;
        }
