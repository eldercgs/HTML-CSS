Seletor

O ID é usado quando temos dois elementos iguais, mas queremos duas formatações diferentes.

Podemos inserir um ID dentro da tag desejada, peguemos , por exemplo o primeiro título do site, por ele ser o título principal podemos querer uma formatação diferenciada dos demais títutlos do site, nesse caso, colocaremos um identificador nele. Veja exemplo:

        h1 id="principal">Seletor  /h1

Em seguida, nas folhas de estilo, (CSS) adicionamos o H1 com modificador.

        h1#principal {
            text-align: center;
            font-size: 20px;
            font-weight: 600;
            text-decoration: underline;
        }

Observção

Tudo que em HTML é id em CSS é # (cerquila). Tudo que me HTML é class em CSS é .(ponto.)

Usando CLASS

Pelas regras da W3C, só podemos acrescentar um ID no HTML, então como fafemos para fazermos as demais formatações? Usaremos uma propriedade chama CLASS que basicamente tem a mesma função do ID, mas pode ser usada mais de uma vez. Veja o exemplo de como fica nas CSS: 

            .basico {
                color: rgb(85, 201, 8);
                font-size: 24px;
                font-family:Verdana, Geneva, Tahoma, sans-serif
            }

Podemos aplicar duas CLASS a mesma tag. Veja exemplo:

        h2 class="basico destaque" /h2

Utilizando mais de uma CLASS

Duas CLASS podem ser colocados dentro das mesmas aspas e não precisam ser separadas por sílabas.

SPAN

A tag SPAN serve para que possamos envelopar elementos do texto, a fim de formatá-los. Veja no código:

        Molestiae quis voluptate autem. Iure, span class="basico"> dolorem  /span> maxime exercitationem ipsa veniam.
Foi escolhida uma palavra aleatória no texto (dolorem) e atribuida a ela uma classe. Assim a mesma acaba herdando a formatação da class BASICO na folha de CSS. 
        .basico {
            color: rgb(85, 201, 8);
            font-size: 24px;
            font-family:fantasy, Geneva, Tahoma, sans-serif;
            background-color: azure ;
        }

ID e CLASS

Podemos, também, colocar um ID e um CLASS na mesma tag. Exemplo:

        h1 id="principal" class="destaque" >Seletor /h1>

Neste caso, declara-se a CLASS depois do ID separadamente, não é necessário vírgula.
