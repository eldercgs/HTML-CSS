Uma das preocupações em se usar o iframe é a segurança. Referente a coleta de dados por sites de terceiros, podemos proteger o iframe adicionando dentro da tag SANDBOX="SANDBOX", assim o botão de ação do formulário ficará inativo.

        iframe src="pag004.html" sandbox="sandbox"  >Seu navegador não é compátivel com isso. /iframe>

podemos adicionar REFERREPOLICY="NO-REFERRER",onde dizemos que o conteúdo que está carregado dentro do iframe não irá coletar qualquer dado do usuário.

        
            iframe src="pag004.html" sandbox="sandbox" referrerpolicy="no-referrer"  >Seu navegador não é compátivel com isso. /iframe>

Para mais detalhes qauntoa autorização de contúdos do iframe dentro site, consultar aula 09 do módulo 4 de CSS3
