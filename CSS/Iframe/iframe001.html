Podemos colocar dentro do iframe um link com o endereço do site que desejamos para que seja visualizado, caso o navegador não suporte o iframe.

Podemos configurar a altura e largura do iframe. Por padrão é 300px por 150px.

As CSSs são soberanas nas configurações. Podemos configurar o iframe dentro da Tag, porém havendo ambas as configurações valerá o CSS.

Podemos configurar a barra de rolagem dentro da tag através do SCROLLING "", onde dentro das aspas, colocaremos a configuração desejada. YES para barra de rolagem (padrão) ou NO para exibir o iframe sem a barra de rolagem. Pode ser que não funcione em todos os navegadores.
