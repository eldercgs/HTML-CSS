Para usar o CSS de maneira externa, a fim de se consiga editar várias páginas simultaneamente, acrecenta-se o LINK:CSS no Head do arquivo. No HREF aparecerá a sugestão de nome de arquivo CSS (style.css), basta deixar o cursor do mouse em cima do nome e precionar CTRL + CLIK e um novo arquivo CSS será aberto.
