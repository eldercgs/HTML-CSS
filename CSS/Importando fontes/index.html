Usando Google fontes

Podemos importar fontes externas através do site https://fonts.google.com/

Lá podemos pesquisar o tipo de fonte, depois de selecionar a fonte, podemos clicar no "quadrado com um pontinho vermelho localizado no canto superior direito da tela e selecionamos IMPORT, logo após copiamos o código sem a tag STYLE e colamos na configuração de estilo do site. Deve ser colado como primeira configuração do style. Veja exemplo:

        
            style
        @import url('https://fonts.googleapis.com/css2?family=Kaushan+Script&display=swap');
        body {
            font-family: Arial, Helvetica, sans-serif;
            font-size: 16px;
            color: darkblue;
        }
        h1 {
            font-family: 'Times New Roman', Times, serif;
        }
        style

Em seguida substituímos o FONT-FAMILY pela fonte importada e atualizamos a página.

        
            style>
        
        @import url('https://fonts.googleapis.com/css2?family=Kaushan+Script&family=Work+Sans:ital,wght@0,100;0,300;1,300&display=swap');
        body {
            font-family: 'Work Sans', sans-serif;
            font-size: 16px;
            color: darkblue;
        }
            /style
