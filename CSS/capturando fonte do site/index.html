Capturando fontes

Você acessa o Google Web Store e pesquisa por "fontes ninja", clica em usar no Chrome e adiciona a extensão. Podemos desligar as extensões pelo caminho mais ferramentas -> extensões -> clicar no botão para ativar ou desativar.

Ao usar a extensão "fonte Ninja", a extensão analisa os tipos de fontes, assim como tamanho e outras especificações e te diz quais são, a fim de que você possa usar a fonte correspondente.

Detectando fontes em imagens

Podemos através do site whatfontis.com,

O site "fonte Squirrel,

No site "myfonts.com"

Para todos os sites, o uso é bem intuitivo, basta arrastar a imagem para o local indicado e o site analisará as possíveis fontes existentes e vai sugerir quais podem ser. Depois é só procurar adquirir a fonte indicada ou pelo menos parecida, vista que a análise nem sempre é 100%, depende, também da qualidade da imagem.
