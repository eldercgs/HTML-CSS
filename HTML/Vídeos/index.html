Vídeos

É possível baixar vídeos sem problemas com direitos autorais no site pexels na aba vídeos.

Deve-se evitar colocar vídeos muito pesados no site, segue-se a mesma premissa das imagens. É possível reduzir o tamanho do vídeo, com um qualidade razoável, baixando o aplicativo HANDBREAK, onde pode-se converter os vídeos para vários formatos desejados.

O peso do vídeo está diretamente ligado ao valor investido na hospedagem do site, podendo gerar muito gasto para o cliente.

Incorporação de vídeos

Nós podemos adicionar vídeos no site linkando-os de plataforma de vídeos na internet.

Por exemplo, podemos usar o YouTube. Escolhemos o vídeo, clicamos em compartilhar e depois em incorporar, esta opção gera um código HTML automaticamente para seu código (é possível modificá-lo), depois é só copiar e colar no mesmo.

O Site Vimeo, também é uma alternativa para busca de vídeos, também é possível utilizar um modo profissional (Business), o qual permite você direcionar os vídeos para pessoas escolhidas, como assinantes do seu curso, por exemplo.
