Usando link externo

Você pode acessar meu facebook

Usa-se a tag A para evidenciar um link, no atributo REL coloca-se o endereço externo.

Você pode, também, acessar meu Instagram

Usamos a tag TARGET com o direcionamento "_blank" para indicar que o link abra em outra aba e adicionalmente podemos usar a tag REF para referenciar ao buscador que esse é um link externo, caso o contrário o link abrirá na mesma página o que pode fazer com que o usuário não volte mais para o seu site.

Link interno

Está é a primeira página do site, se vc quiser pode acessar a segunda página.

Para colocar um link interno basta usar a tag A e referênciar a página para onde quer que o link abra.

Podemos o usar a tag REL NEXT para indicar ao buscador que o link encaminha para a próxima página.

No atributo REF é possível usar o atalho CONTROL +ESPAÇO.
